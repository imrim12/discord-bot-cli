# discord pins

List pinned messages in a channel.

## Usage

```bash
discord pins "#general"
discord pins 1473974057459384404
```

## Arguments

| Arg | Description |
|-----|-------------|
| `channel` | Channel ID or `#channel-name` |

## Options

| Flag | Description |
|------|-------------|
| `-g, --guild <id>` | Guild ID for resolving `#channel-name` |

## When to use

- Find important/reference messages in a channel
- Review pinned decisions, links, or announcements
